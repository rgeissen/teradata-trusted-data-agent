Key Features

A powerful toolkit for AI-driven data exploration and analysis.

Multi-Provider LLM Support

Dynamically switch between Google, Anthropic, AWS Bedrock, and local Ollama models to compare reasoning and performance in real-time.

Unmatched Transparency

The Live Status panel offers a real-time window into the AI's mind, showing its thought process, tool calls, and raw data for ultimate clarity and trust.

System Prompt Editor

Take full control of the agent's behavior by editing, saving, and resetting the core system prompt that guides its actions and personality.

Integrated Charting Engine

Go beyond tables. The agent can automatically generate insightful charts and visualizations from your data to reveal trends and patterns.

Dynamic Capability Loading

The agent automatically discovers and integrates all available tools and prompts from the connected Teradata MCP Server, ensuring it's always up-to-date.

Token Usage Tracking

Monitor the cost and efficiency of your interactions with detailed token counts for every LLM call, helping you optimize prompts and usage.

The agent uses a sophisticated workflow to translate your questions into answers. It starts with the **Frontend UI** capturing your request. The **Backend Server** then orchestrates the process, sending your request and the available tools to the **Large Language Model (LLM)** for reasoning.

The LLM intelligently selects the correct tool and parameters, which the backend then executes against the secure **Teradata MCP Server**. The results are returned to the LLM for synthesis, and the final, human-readable answer is streamed back to the UI.
